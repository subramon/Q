This directory contains different python file which uses sklearn DT algo for each dataset (titanic, breast_cancer, ramesh dataset).

File name contains the dataset information.

Below files are used when you have train and test dataset in separate file
    DTree_sklearn_breast_cancer_train_test.py
    DTree_sklearn_ramesh_dataset_train_test.py
    DTree_sklearn_titanic_train_test.py

Here, you need to specify the csv file path in script.

Command to run the python code
$ cd Q/ML/DT/python
$ python -Wignore <file_name>

Example:
$ python -Wignore DTree_sklearn_breast_cancer.py
